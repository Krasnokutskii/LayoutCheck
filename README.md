# LayoutCheck

This app helps understand how layout works. Visual format and Anchors



## Tools implemented in project

- translatesAutoresizingMaskIntoConstraints

- addSubview()

- NSLayoutConstraint.constraints(withVisualFormat:)

- :H

- :V
